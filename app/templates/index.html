About

Hi! My name is Bobby and I own this domain. This website was made entirely from scratch! You can look at the source code here! It's quite simple, for now. Sometimes it changes when I'm feeling ambitious!

This website was built using Python and the Flask web framweork. The Flask app is run with Apache on a Google Compute Engine virtual machine. The virtual machine is so small, it only has 600mb of RAM! The VM costs about $4.39 a month to run. In cojunction with the $12.00/year I pay for the domain name, this website costs me about $65 a year to keep up. That's less than I pay for groceries every month!
